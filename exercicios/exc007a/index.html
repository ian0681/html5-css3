As principais formatações

Negrito

Para formarmos uma palavra ou frase em negrito nós temos que utilizar <b>"texto desejado"</b>
Outra forma de se destacar um texto e utilizando a tag <strong>"texto desejado"</strong>

Da primeira forma o termo apenas fica destacado, já da segunda forma o termo destacado ganhar um valor semântico

Itálico

Para se colocar um texto em itálico, se utiliza a tag <i>"texto desejado"</i> no entanto desta forma não a significado/valor semântico par o html.
Para se colocar um texto em itálico com significado para o html, se utiliza a tag <em>"texto desejado"</em>

Texto Marcado

Para marcar um texto como se fosse um marca texto, podemos utilizar a tag <mark>"texto desejado"</mark>

Texto deletado

Podemos riscar um texto com a tag <del>"texto desejado"</del>

Sublinhado

Podemos Sublinhar um texto utilizando a tag <ins>"texto desejado"</ins> dessa forma atribuimos um valor semântico ao texto, mas também podemos utilizar a tag <u>"texto desejado"</u>, no entanto ela não possue valor semântico
